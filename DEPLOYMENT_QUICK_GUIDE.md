## 🔍 4 דרכים למצוא את Deploy

### דרך 1: כפתור מרכזי (הכי פשוט!)
```
אחרי שבחרת את הפרויקט
→ תראה כפתור גדול: "🚀 Deploy Now"
→ לחץ עליו!
```

### דרך 2: Settings
```
Settings → Deployments → Trigger Manual Deployment
```

### דרך 3: תפריט עליון
```
☰ (המבורגר) → Deploy
```

### דרך 4: Integrations
```
Settings → Integrations → GitHub → Sync Now
```

## 🎯 איפה הכפתור?

הכפתור יכול להיות ב-**3 מקומות**:

```
📍 מקום 1: באמצע הדף (הכי נפוץ!)
   [🚀 Deploy Now] ← כפתור כחול/ירוק גדול

📍 מקום 2: למעלה מימין
   [...] [🚀 Deploy] [⚙️] [👤]

📍 מקום 3: בתפריט צד
   ☰ Menu
     → 🚀 Deploy
```

---

## ❌ אם אתה לא מוצא את הכפתור

### נסה את זה:
1. ✅ רענן דף (F5)
2. ✅ בדוק שבחרת את הפרויקט "אפריון"
3. ✅ חפש בתפריט Settings
4. ✅ חפש כפתור "Redeploy" במקום "Deploy"
